## Install
```bash
git clone https://github.com/swumao/exam-enroll.git
cd exam-enroll
composer install # 如果没有安装composer需要先安装
# 然后配置apache虚拟主机，web根目录指向exam-enroll/web/htdocs
# Congratulation!
```
